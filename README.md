# 🔐 Sign Up / Login System

A front-end authentication flow with registration, login and a protected home page, built with vanilla JavaScript and form validation, using localStorage as the user store.

## 📖 Overview

A complete client-side auth experience: users can register, sign in with their credentials, land on a personalized home page, and log out. User accounts are validated with regular expressions and stored in the browser's **localStorage**.

## ✨ Features

- 📝 **Registration** with real-time input validation (name, email, password) via RegEx
- 🔑 **Login** with credential checking against saved users
- 👋 **Protected home page** that greets the logged-in user by name
- 🚪 **Logout** that clears the session
- 💾 Users persisted in localStorage

## 🛠️ Tech Stack

- HTML5 & CSS3
- Bootstrap
- Vanilla JavaScript (ES6+, RegEx validation, localStorage)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/Signup-Login.git
cd Signup-Login
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).

> ℹ️ This is a front-end demo — authentication is simulated in the browser and is **not** intended for production use.
